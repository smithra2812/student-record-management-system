# Student Record Management System

## 📌 Project Overview

This project is a basic Student Record Management System developed to efficiently manage student data.

## 🚀 Features

* Add student details
* View student records
* Simple data storage using lists
* Menu-driven program

## 🛠 Technologies Used

* Python (Basic)
* DBMS Concepts

## 📊 Outcome

This project helped in understanding how structured data is managed using programming logic and basic database concepts.

## 💡 Future Improvements

* Integration with MySQL database
* GUI interface using Tkinter
* Deployment using cloud platforms like AWS
